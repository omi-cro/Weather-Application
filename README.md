# Simple Weather Application

This is a console-based Weather Application that allows users to enter a city name and view current weather details.

## Features
- Fetches real-time weather data using OpenWeatherMap API
- Displays temperature and weather condition
- Uses API calls and JSON parsing

## Technologies Used
- Python
- OpenWeatherMap API
- Requests library

## How to Run
1. Install Python
2. Install requests library:
   pip install requests
3. Add your API key in weather.py
4. Run:
   python weather.py
